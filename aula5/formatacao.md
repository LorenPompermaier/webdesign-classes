# Resumo

> Nessa aula iremos ver mais **tags** do HTML, para compreender como elaborar mais nossos documentos.

### `<bold> Negrito </bold>`

O elemento `<strong>` também representa uma seção de texto que seria apresentar de uma maneira visualmente diferente (por exemplo, palavras-chave em um parágrafo).


```
 <strong> Sou uma palavra em negrito </strong>
 <em> Sou uma palavra em itálico </em>

```

### `<em>` Itálico `</em>`

O elemento `<em>` também representa uma seção de texto que seria dita de maneira diferente do conteúdo ao redor - como termos técnicos, palavras estrangeiras, pensamentos ou outros termos que normalmente seriam em itálico.

```
 <em> Sou uma palavra em itálico </em>
```


### `<sup>`
```
O elemento `<sup>` é usado
para conter caracteres que devem estar sobrescritos,
como sufixos de datas ou conceitos matemáticos, como elevar um número a um poder como 2ˆ2.
```
> Exemplo:

- 2<sup>2</sup> = 4
- 2<sup>3</sup> = 8
- f(x)=2<sup>x</sup>
- f(x)=log<sup>x</sup><sub>2</sub>

### `<sub>`

```
O elemento `<sub>` é usado para conter caracteres que devem ser subscritos. É comumente usado com notas de rodapé ou fórmulas químicas como o H20.

```
> Exemplo:

H<sub>2</sub>O

### Quebra de linha `<br>`

 Se você quiser adicionar uma quebra de linha no meio de um parágrafo, use a tag `<br />`.

```
 <p> Aqui tem um texto onde que iremos quebrar linha aqui <br>
 e aqui, <br>
 é assim que funciona a tag.

```

### `<hr>`

Para criar uma pausa entre temas, como uma mudança de tópico em um livro ou uma nova sessão
da nossa página - você pode adicionar uma linha horizontal entre as seções usando a tag `<hr>` 

Existem alguns elementos que não têm palavras entre uma tag de abertura e fechamento. Eles são conhecidos como elementos vazios e são escritos de maneira diferente.

> Exemplo:

```
<h4>Requisitos minímos:</h4>
<hr />
 <ul>
  <li> Item A</li>
  <li> Item B</li>
  <li> Item C</li>
 </ul>

<h4>Diferenciais:</h4>
<hr />

 <ul>
  <li> Item 1</li>
  <li> Item 2</li>
  <li> Item 3</li>
 </ul>

```
